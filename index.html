Sum of a list
Calculate the sum of a list.
The sum of an empty list is generally defined as 0.
Iterative:
We get the sum of a list by initializing a variable to zero and then loop through the list adding each element to that variable. 

def sum(xs):
    if len(xs) == 0:
        return 0

    result = 0

    for x in xs:
        result += x

    return result
Concatenative:
The sum of the list is a reduction of that list using the addition function with an initial value of zero. 

sum := reduce {add} 0
